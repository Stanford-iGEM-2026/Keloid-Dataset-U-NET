Segmented Keloid Dataset -- SLAAAAY

The dataset annotations were originally provided by CVAT (Computer Vision Annotation Tool) in an annotations.xml file. Since CVAT's mask export features require a paid subscription lol, additional code was developed to convert the coordinate-based annotations into PNG segmentation masks compatible with U-Net training pipelines.
